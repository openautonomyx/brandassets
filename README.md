# AutonomyX Brand Assets

**Open Enterprise Innovation**

AutonomyX is a Framework First enterprise innovation platform built on a Trusted Stable Substrate.

- Stable Substrate

- OpenTofu as Infrastructure as Code
- Operator Framework as Reconciliation
- Kubernetes and OpenShift as Execution
- Open Metal as the Base
