## Positioning

AutonomyX is an Open Metal-based Framework First enterprise innovation platform built on a Stable Substrate, where Trust is a Primitive, Proto defines contracts, Fabric is the Stable Graph, Rust is the Kernel Core, SurrealDB is the runtime provider, OpenTofu defines infrastructure, and Operator Framework reconciles lifecycle across Kubernetes and OpenShift.

## Definitions

- **Open Metal:** sovereign infrastructure base
- **Stable Substrate:** durable foundation for trust, contracts, graph, runtime, infrastructure, and reconciliation
- **Trust Primitive:** identity, policy, provenance, audit, verification, and control built into the core
- **Framework First:** architecture, contracts, extension points, governance, and lifecycle are defined before implementation
- **Proto as Contract:** machine-readable contracts for kernel, graph, runtime, APIs, SDKs, agents, and integrations
- **Stable Fabric Graph:** versioned, governed, testable, portable, and backward-compatible enterprise relationship model
- **Rust Kernel Core:** trusted execution foundation
- **SurrealDB:** runtime provider for the Stable Fabric Graph
- **OpenTofu:** infrastructure as code
- **Operator Framework:** reconciliation and lifecycle engine
- **Kubernetes:** orchestration substrate
- **OpenShift:** enterprise adoption path
